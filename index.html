    Work, and Work in Progress...

    Go to project:

    • Flickr Clone

      This app uses Ruby on Rails and Javascript (AJAX calls) to manage photo uploading and sharing between friends. Multiple photos can be uploaded simultaneously and can put into collections and have taggings that can be searched for in the search bar. People can then see your photos and add them to favorites and also add you as friend to see more of your photos on a feed. All photos are stored and served by Amazon S3 services.

    • Meme Maker

      This is a fun, simple app that users can insert meme captions into their favorite meme templates and link the newly generated memes to their friends. Templates can either be chosen from a pre-populated list or can be uploaded by the user. This app uses pure backbone in the front end and rails as a server in the back end. All photos are stored and served by Amazon S3 services.

    • Asteroids Game

      The classic Asteroids game from the 1970s now recreated in Javascript on Canvas. This project uses pure javascript logic to draw onto a canvas, where a user can navigate a ship to shoot down asteroids while avoiding collision.
